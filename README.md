  Agenda de Contatos em Python

Projeto desenvolvido durante meus estudos de **Python**, com o objetivo de praticar conceitos básicos da linguagem como:

* Funções
* Listas e dicionários
* Estruturas condicionais
* Loops
* Manipulação de dados no terminal

Funcionalidades

O programa funciona via **terminal** e permite:

* Adicionar contatos
* Visualizar contatos
* Editar contatos
* Marcar / desmarcar contato como favorito
* Visualizar contatos favoritos
* Deletar contatos

Conceitos praticados

Durante o desenvolvimento foram utilizados conceitos como:

* Estruturas de dados (`list`, `dict`)
* Funções
* `for` e `while`
* `enumerate`
* Manipulação de listas
* Organização de código em funções
